# CompGenomics-metaGEO

## R package requirements
It is recommended to install these packages via Bioconductor's biocLite() method:
* shiny
* shinyjs
* shinyBS
* lattice
* logging
* hgu133plus2.db
* org.Hs.eg.db
* knitr
* gdata
* dbConnect
* RSQLite
* GEOmetadb
* xtable
* rmarkdown

### Run example metaGEO shiny-app:
```R
mypackage::runExample()
```
